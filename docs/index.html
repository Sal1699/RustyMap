▸ Install da terminale

One-liner che rileva OS/arch, scarica l'ultimo binario dalla release, verifica le dipendenze e lo installa in PATH.

Linux / macOS

λ curl -fsSL https://sal1699.github.io/RustyMap/install.sh | bash

Default: installa in /usr/local/bin (o ~/.local/bin senza sudo). Per cambiare: | bash -s -- --prefix=$HOME/.local/bin. Per versione specifica: --version=v0.1.0.

Windows (PowerShell)

λ iwr -useb https://sal1699.github.io/RustyMap/install.ps1 | iex

Installa in %LOCALAPPDATA%\Programs\RustyMap e aggiorna il PATH utente. Npcap va installato a parte con rustymap --install-npcap (admin).

Cargo (da crates.io, quando pubblicato)

λ cargo install rustymap

Richiede Rust 1.70+ e libpcap-dev su Linux.

▸ Installazione manuale (archivio)

Windows (PowerShell admin)

λ Expand-Archive rustymap-windows-x86_64.zip -DestinationPath .
λ cd rustymap-x86_64-pc-windows-msvc
λ .\rustymap.exe --guide

Per scan SYN/raw installa Npcap: .\rustymap.exe --install-npcap

Linux / macOS

λ tar xzf rustymap-linux-x86_64.tar.gz
λ cd rustymap-x86_64-unknown-linux-gnu
λ sudo ./rustymap --guide

Raw scans richiedono sudo o CAP_NET_RAW.

Da sorgente (qualsiasi OS)

λ git clone https://github.com/Sal1699/RustyMap.git
λ cd RustyMap
λ cargo build --release
λ ./target/release/rustymap --guide

Richiede Rust 1.70+ e libpcap-dev su Linux.

▸ Prime scansioni

# TCP connect scan (no privilegi)
λ rustymap --sT 10.0.0.0/24

# SYN scan stealth con fingerprint Windows 11
λ rustymap --sS --evasion stealth --stack-profile windows11 10.0.0.5

# Audit completo con report HTML
λ rustymap --sS --sV -O --oH report.html 10.0.0.0/24

# Idle scan via zombie
λ rustymap --sI 192.168.1.250:80 10.0.0.5

# Lista completa comandi ed esempi
λ rustymap --guide
